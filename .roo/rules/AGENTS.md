# 项目规则 - 打开股票app

本项目要求实现:
- 提供api接口供 飞书多维表格里的按钮 来调用
- 收到http请求, 打开聚焦搜索(最好能在后台打开),
- 搜索栏填入http请求带入的股票名参数, 然后回车
- 其实这样腾讯自选股会是第一个响应的, 这样就实现了一键看股票行情

## 个人要求

- 项目必须在 mac 环境下运行
- 项目在 Python 3.12 环境下运行
- 代码极简主义, 命名和实现都应越简单越好
- 代码写紧凑一点, 函数间空一行! 函数内不能有空行!
- 不要自动创建代码修改后总结的md, 我需要的话会问你的
- import统一放到文件顶部, 不要在函数内部导入
- 不用直接用python和pip和pm2, 而要通过uv来用
- 所有的环境变量以.env为准, 请无视.env.example
- api请求都用POST
